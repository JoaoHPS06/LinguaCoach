As funções mais importantes, por categoria
Texto e títulos

st.title() — título grande da página
st.header() / st.subheader() — subtítulos
st.write() — texto genérico, aceita strings, dicts, dataframes
st.markdown() — texto com formatação markdown e HTML

Inputs do usuário

st.text_area() — caixa de texto grande (o seu caso)
st.text_input() — campo de texto de uma linha
st.selectbox() — dropdown de seleção
st.button() — botão clicável
st.checkbox() — caixa de marcação
st.slider() — controle deslizante

Exibição de resultados

st.success() — caixa verde de sucesso
st.error() — caixa vermelha de erro
st.warning() — caixa amarela de aviso
st.info() — caixa azul informativa
st.expander() — seção expansível/colapsável
st.metric() — exibe um número com destaque

Layout

st.columns() — divide a tela em colunas lado a lado
st.sidebar — painel lateral
st.tabs() — abas navegáveis
st.divider() — linha separadora

Estado e sessão

st.session_state — dicionário que persiste dados entre reruns

Gráficos

st.bar_chart() — gráfico de barras simples
st.line_chart() — gráfico de linha


Como o Streamlit executa o código:
Esse é o ponto mais importante de entender: toda vez que o usuário interage com qualquer elemento, 
o script inteiro roda do zero, de cima para baixo. Por isso o input() do terminal não funciona — 
e por isso o st.session_state existe, para você preservar informações entre essas execuções.
Para o LinguaCoach você vai usar principalmente: text_area, selectbox, button, expander, markdown, 
session_state e bar_chart. Cobre praticamente tudo do roadmap.
